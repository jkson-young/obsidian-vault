# 活动助手

## 简介

活动助手是盛天网络开发的网吧/门店管理客户端产品，为线下门店提供数字化运营工具。

## 系统架构

活动助手采用 C/S 架构：

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  ZhanBa.exe │────→│ zhanba_cli  │────→│ zhanba_ser  │
│   (客户端)   │     │  (CLI程序)  │     │  (服务端)   │
└─────────────┘     └─────────────┘     └──────┬──────┘
                                                │
                                                ↓
                                          ┌─────────────┐
                                          │    DSSP     │
                                          │  (后端服务)  │
                                          └─────────────┘
```

## 核心组件

| 组件 | 说明 |
|------|------|
| ZhanBa.exe | 活动助手主程序，用户操作界面 |
| NetBarInfo.exe | 网吧信息识别模块 |
| zhanba_ser | 服务端程序，处理业务逻辑 |
| zhanba_cli | CLI 客户端，连接服务端的代理程序 |
| zhanba_ser_login | 服务端登录器 |

## 部署模式

- **生产环境**: 正式域名，线上数据
- **测试环境**: 独立域名，通过修改配置文件和 host 绑定切换
