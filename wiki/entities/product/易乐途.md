# 易乐途

## 简介

易乐途（Yiletoo）是盛天网络开发的网吧/门店管理系统，为线下门店提供数字化运营管理解决方案。

## 系统架构

易乐途采用前后端分离架构：

```
┌─────────────────────────────────────────────────────────┐
│                        前端服务                          │
├─────────────┬─────────────┬─────────────┬───────────────┤
│ admin       │ yun         │ desktop     │ cashier-ui    │
│ (管理后台)   │ (云服务)     │ (客户端桌面) │ (收银台)       │
├─────────────┴─────────────┴─────────────┴───────────────┤
│                        后端服务                          │
├─────────────┬─────────────┬─────────────┬───────────────┤
│ api         │ diskless    │ openapi     │ server        │
│ (主API)     │ (无盘服务)   │ (开放平台)   │ (服务端)       │
├─────────────┴─────────────┴─────────────┴───────────────┤
│                      客户端程序                          │
├─────────────────────────┬───────────────────────────────┤
│ yiletoo_Server.exe      │ yiletoo_Client.exe            │
│ (服务端)                 │ (客户端)                       │
└─────────────────────────┴───────────────────────────────┘
```

## 核心组件

| 组件 | 说明 |
|------|------|
| yiletoo_Server | 易乐途服务端程序 |
| yiletoo_Client | 易乐途客户端程序 |
| admin.yiletoo.com | 管理后台 |
| api.yiletoo.com | 后端 API 服务 |
| desktop.yiletoo.com | 客户端桌面服务 |
| cashier-ui.yiletoo.com | 收银台界面 |
| openapi.yiletoo.com | 开放平台接口 |

## 与途邦SCRM集成

易乐途作为设备数据源，通过途邦SCRM将门店设备信息上报至美团等平台。

**数据链路**:
```
易乐途门店设备 → 途邦SCRM → 美团
```
