# 途邦SCRM

## 简介

途邦SCRM是盛天网络的门店客户关系管理系统，核心能力包括：
- 接收易乐途等门店系统的设备数据
- 数据清洗和规则过滤
- 多平台数据对接（美团等）

## 系统架构

```
┌─────────────────┐
│   易乐途门店     │
│   (设备数据源)   │
└────────┬────────┘
         │ 设备数据上报
         ↓
┌─────────────────┐     ┌─────────────────┐
│   途邦 SCRM     │────→│   数据清洗规则   │
│  (数据中间层)    │     │  (过滤无效数据)  │
└────────┬────────┘     └─────────────────┘
         │ 清洗后数据存储
         ↓
┌─────────────────┐
│    数据存储层    │
└────────┬────────┘
         │ 第三方平台查询
         ↓
┌─────────────────┐
│  美团/其他平台   │
└─────────────────┘
```

## 核心功能

| 功能 | 说明 |
|------|------|
| 数据接收 | 接收门店系统设备上报数据 |
| 数据清洗 | 通过规则过滤，决定哪些数据可上报 |
| 平台对接 | 被动响应第三方平台的数据查询 |
