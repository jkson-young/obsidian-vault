## 技术要点

### 核心组件

| 组件 | 说明 |
|------|------|
| ZhanBa.exe | 活动助手主程序（用户界面） |
| NetBarInfo.exe | 网吧信息识别模块 |
| zhanba_ser.exe | 服务端程序 |
| zhanba_cli.exe | CLI 客户端 |
| zhanba_ser_login.exe | 服务端登录器 |

### 配置文件

- `config/app.ini` - 主配置文件（包含服务器地址）

### 注册表位置

- `HKEY_LOCAL_MACHINE\SOFTWARE\zhanba_cli` - 存储登录门店信息
