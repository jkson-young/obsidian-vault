## 数据链路验证

```
易乐途门店设备 → 途邦SCRM（数据清洗）→ 存储 → 美团查询 → 数据同步
```

### 验证步骤

1. 易乐途设备上报数据
2. 检查途邦是否接收到数据
3. 检查数据清洗规则是否生效
4. 调用美团查询接口验证数据同步
