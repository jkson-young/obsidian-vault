# 途邦SCRM-易乐途门店设备信息上报美团测试

> 原始资料：[[raw/途邦SCRM-易乐途门店设备信息上报美团测试|途邦SCRM-易乐途门店设备信息上报美团测试]]

## 概述

本文档描述测试环境模拟易乐途门店设备信息上报美团的完整流程。

## 数据上报链路

```
易乐途门店设备
    ↓（开机/关机/重启时上报）
途邦 SCRM
    ↓（数据清洗规则过滤）
美团平台
    ↓（美团主动调用查询接口）
同步设备信息
```

> **注意**: 途邦不会主动给美团上报数据，美团调用接口查询门店设备信息时，数据才会同步给美团。

## 测试步骤

### 1. 安装易乐途测试环境

参考 [[sources/易乐途测试环境安装]] 完成环境搭建。

- 登录服务端后即可调用设备上报接口模拟门店设备数据上报
- 打开客户端模拟设备启动时上报数据
- 关闭客户端模拟设备关闭时上报数据

### 2. 易乐途门店绑定美团门店

**方式一：途邦管理平台授权**

通过"授权美团门店"功能绑定。

> 测试环境美团门店账号：
> - 账号：`develop_tuangou304`
> - 密码：`m2QjWr`

**方式二：数据库直接操作**

在 `tobang` 数据库 `middle_office.mt_shop_auths` 表中操作：

| 字段 | 说明 |
|------|------|
| is_auth | `1` = 绑定，`0` = 解绑 |

### 3. 调用易乐途设备上报接口

**接口信息**

- URL：`http://192.168.58.149:52745/pms_PostDeviceInfo`
- Query 参数：`shopid=90007210`
- Content-Type：`application/json`

**请求 Body 示例**（设备全量信息）：

```json
{
    "audio": [...],
    "baseboard": {...},
    "cpu": [...],
    "disks": [...],
    "display_resolution": "width:1920,height:1080",
    "gpu": [...],
    "hids": [...],
    "ip": ["192.168.150.100"],
    "mac": "E8-9C-25-37-9E-9A",
    "memories": [...],
    "monitor": [...],
    "name": "机器1",
    "netcard": [...],
```

> 完整字段详见原始资料。

**数据展示规则**

| 绑定状态 | 数据展示位置 |
|---------|-------------|
| 未绑定美团门店 | 未授权列表 |
| 已绑定美团门店 | 已授权列表 |

### 4. 调用美团查询接口

**接口信息**

- URL：`https://tb-md-t.stnts.com/api/meituan/v1/callback/callback`
- Content-Type：`x-www-form-urlencoded`

**请求参数**

| 参数 | 值 | 说明 |
|------|-----|------|
| msgType | `5810155` | 消息类型 |
| developerid | `117090` | 开发者ID |
| sign | `c339bc79e5cba76d71798b7bbb938edaebd140ee` | 签名（找开发索要） |
| msgid | `7668614585338609882` | 消息ID |
| timestamp | `1778861127` | 时间戳 |
| businessId | `58` | 业务ID |
| message | `{"opPoiId":"A13HE4SUAO93DVI01RJ1U9O9PKT4MOGM6K4A024BDOVFE4JS21L10","pageSize":50,"pageNo":1}` | 业务数据（含美团门店ID） |

**调用效果**

调用该接口可查询美团门店绑定的易乐途门店下的全量设备信息，此时易乐途上报给途邦的数据会同步给美团。

## 关键概念

| 概念 | 说明 |
|------|------|
| opPoiId | 美团门店唯一标识 |
| gid / shop_id | 易乐途门店标识 |
| 数据清洗规则 | 途邦SCRM管理平台配置的过滤规则 |
| 全量设备信息 | 包含CPU、显卡、内存、硬盘、显示器等完整硬件信息 |

## 测试数据状态流转

```
设备操作（开机/关机/重启）
    ↓
易乐途上报设备信息到途邦
    ↓
途邦数据清洗 → 存储
    ↓
美团调用查询接口
    ↓
途邦将数据同步给美团
```
