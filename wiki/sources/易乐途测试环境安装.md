# 易乐途测试环境安装

> 原始资料：[[raw/易乐途测试环境安装|易乐途测试环境安装]]

## 概述

本文档描述易乐途客户端连接测试环境的完整配置流程。

## 安装步骤

### 1. 安装易乐途服务端

安装程序：`yiletoo_Server.exe`

### 2. 安装易乐途客户端

安装程序：`yiletoo_Client.exe`

### 3. 配置 Hosts

在 `C:\Windows\System32\drivers\etc\hosts` 中绑定易乐途测试环境域名：

```
# 易乐途-前端-测试1
10.0.45.159 admin.yiletoo.com
10.0.45.159 yun.yiletoo.com
10.0.45.159 desktop.yiletoo.com
10.0.45.159 chat-ui.yiletoo.com
10.0.45.159 cashier-ui.yiletoo.com
10.0.45.159 service-desk-ui.yiletoo.com

# 易乐途-后端-测试1
10.0.45.159 api.yiletoo.com
10.0.45.159 diskless.yiletoo.com
10.0.45.159 openapi.yiletoo.com
10.0.45.159 openapi-test1.yiletoo.com
10.0.45.159 beyondh-test.yiletoo.com
10.0.45.159 server.yiletoo.com
10.0.45.159 beyondh.yiletoo.com
10.0.41.140 account.yiletoo.com
```

### 4. 启动服务端并登录

- 账号：`10160001`
- 密码：`st142536`

### 5. 启动客户端

客户端启动成功后，服务端客户机状态会+1。

## 关键概念

| 概念 | 说明 |
|------|------|
| admin.yiletoo.com | 管理后台域名 |
| api.yiletoo.com | 后端 API 域名 |
| desktop.yiletoo.com | 客户端桌面服务 |
| cashier-ui.yiletoo.com | 收银台 UI |
| openapi.yiletoo.com | 开放平台接口 |
