# 无畏契约赛事数据

## 简介

无畏契约赛事数据是活动助手通过 VALSdk 采集的《Valorant》（无畏契约）电竞比赛相关数据，用于赛事统计、战绩展示和数据分析。

## 数据采集流程

```
┌─────────────────┐     ┌──────────────────┐     ┌─────────────────────┐
│  Valorant 游戏   │────→│   活动助手客户端  │────→│  zhanba_expert_event │
│   (对局进行中)   │     │    (VALSdk)      │     │    (原始事件表)      │
└─────────────────┘     └──────────────────┘     └──────────┬──────────┘
                                                            │
                                                            ↓
                                                   ┌─────────────────┐
                                                   │ esports_match_  │
                                                   │   data_team     │
                                                   │  (对局数据表)    │
                                                   └─────────────────┘
```

## 数据表说明

### zhanba_expert_event

- **用途**：存储活动助手采集的原始游戏事件
- **查询方式**：通过 `uid` + `timeline` 检索
- **数据内容**：原始事件流，包含击杀、死亡、技能使用等

### esports_match_data_team

- **用途**：存储加工后的团队对局数据
- **查询方式**：通过 `match_id` 检索
- **数据内容**：汇总后的对局统计、团队表现数据

## 关键字段

| 字段                 | 说明                     |
| ------------------ | ---------------------- |
| uid                | 用户唯一标识                 |
| timeline           | 时间线（日期标识，如 `20260318`） |
| match_id           | 赛事对局唯一标识               |
| currentTeamPersons | 当前队伍成员列表（含 uid）        |
