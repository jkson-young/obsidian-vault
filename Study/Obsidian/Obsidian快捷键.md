| 快捷键              | 作用                                  |
| ---------------- | ----------------------------------- |
| Ctrl + P         | 打开[[命令面板]]                          |
| Ctrl + 0         | 打开[[快速切换器]]                         |
| Ctrl + S         | 切换堆叠标签页，可让你同时查看多篇笔记                 |
|                  | 当你打开了多个堆叠标签页时，按住Shift键并滚动鼠标滚轮即可横向切换 |
| Ctrl + E         | 从编辑模式切换到阅读模式                        |
| Ctrl + Shift + E | 从编辑模式切换到源代码格式模式                     |
| Ctrl + Click     | 在新标签页中打开笔记                          |

编辑文本操作

| 快捷键                  | 作用                   |
| -------------------- | -------------------- |
| Ctrl + I             | *把文字变成斜体*            |
| Ctrl + B             | **把文字变成粗体**          |
| '##'                 | 文本前添加##+空格，该行会变成二级标题 |
| ~~                   | 文字会添加删除线             |
| **                   | **选中文本输入**，文字会加粗     |
| *                    | *选中文本输入*，文字会变为斜体     |
| Ctrl + 3             | 改为无序列表/有序列表          |
| Ctrl + Enter         | 改为任务列表、勾选完成任务        |
| [[]]                 | 创建链接，新建笔记            |
| ![[test.jpg \| 100]] | 生成一张test.jpg的小尺寸图片   |
| > + 空格               | 标记文本行                |
| ==                   | 文字会变成高亮样式            |
| %%                   | 文字会变成隐藏注释            |
