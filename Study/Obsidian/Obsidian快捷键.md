| 快捷键              | 作用                                  |
| ---------------- | ----------------------------------- |
| Ctrl + P         |                                     |
| Ctrl + 0         |                                     |
| Ctrl + S         | 切换堆叠标签页，可让你同时查看多篇笔记                 |
|                  | 当你打开了多个堆叠标签页时，按住Shift键并滚动鼠标滚轮即可横向切换 |
| Ctrl + E         | 从编辑模式切换到阅读模式                        |
| Ctrl + Shift + E | 从编辑模式切换到源代码格式模式                     |
| Ctrl + Click     | 在新标签页中打开笔记                          |

编辑文本快捷键

| 快捷键      | 作用      |
| -------- | ------- |
| Ctrl + I | 把文本变成斜体 |
| Ctrl + B | 把文本变成粗体 |
