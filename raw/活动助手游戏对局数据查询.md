##### 活动助手无畏契约赛事如何查看对局数据：
1. 对局完成后先查看**zhanba_expert_event**是否收到对局数据上报
- **方式一**： 在梧桐树中使用**自助查询**来查询数据，查询语句：
```
# 通过用户uid查询数据
select * from zhanba_expert_event where uid in (103667905, 103667940) where timeline = '20260318'

# uid可通过客户端赛事详情接口返回数据currentTeamPersons字段中获取
```
![[Pasted image 20260407195637.png]]

- **方式二**：在我的数据中模糊搜索zhanba_export_event表，点击**数据详情**查看每天的数据
![[Pasted image 20260407195942.png]]

2. 再查看对局数据是否推送到zhanba_expert数据库中esports_match_data_team表，查询语句：
```
# 通过赛事id查询数据
select * from esports_match_data_team where match_id = 3182
```

##### 活动助手游戏数据没有成功上报问题本地排查：
1. 是否已替换活动助手最新文件；
2. C:\Windows\System32\drivers\etc\hosts是否已绑定测试环境的ip地址；
3. 查看zhanba安装目录log中的日志文件：
   - 无畏契约赛事相关查看VALSdk*.log
