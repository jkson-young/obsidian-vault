第二大脑（Second Brain / Personal OS）是指一个外部化的知识管理系统，用于存储、组织和检索个人的想法、笔记和项目信息，从而扩展人类的认知能力。

## 核心组成

```mermaid
flowchart LR
    A[输入：写作与反思] --> B[存储：互连的 markdown 文件]
    B --> C[处理：Agent 读取与模式发现]
    C --> D[输出：洞察、想法、行动]
    D --> A
```

## 为什么 markdown 是基础

- **人类可读**：即使不用任何工具，文件本身就有意义
- **完美记忆**：不像人类记忆会扭曲，文件记录的是那个时刻的真实想法
- **Agent 友好**：LLM 以 markdown 为「氧气」，是其最自然的输入格式
- **可移植**：不绑定任何特定平台

## 关键实践

1. **持续写作**：日常反思是生成想法和为 Agent 提供上下文的引擎
2. **建立链接**：通过 [[wikilinks]] 让想法之间产生关联
3. **实体化**：为每个重要的人、项目、概念创建独立页面
4. **人类优先**：vault 中保持人类原创，Agent 输出作为辅助

## 从个人到团队

同样的模式可以扩展到团队：为团队创建一个共享的 Obsidian vault，所有成员贡献上下文，Agent 从中提取洞察和可行的建议。
