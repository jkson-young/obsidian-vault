Tech With Tim 是一位技术教育内容创作者，专注于编程教程、AI 工具使用和开发实践。他以系统化、结构化的教学方式闻名，内容覆盖 Python、AI/ML、Web 开发等领域。

## 核心贡献

- **提示工程的系统化教学**：将提示工程拆解为可学习的技能框架（role/audience/tone/format 四要素、few-shot、chain-of-thought、structured output 等）
- **自然语言编程的类比**：提出 "prompt engineering is just like programming in natural language"，降低了学习门槛
- **Steering vs Commanding 框架**：区分「命令模型」和「引导模型」两种交互模式
- **效率实践**：推广语音 dictation 作为写 prompt 的主要输入方式

## 内容特点

- 实操演示为主，几乎每个技巧都在 ChatGPT 中实时展示
- 提供可下载的参考文档（markdown 格式的 prompt 模板）
- 兼顾初学者和开发者（涵盖 API 参数如 temperature、system vs user prompts）

## 项目

- YouTube 技术教育频道
- 编程课程与教程
- Newsletter（提供 prompt 模板和参考文档）
